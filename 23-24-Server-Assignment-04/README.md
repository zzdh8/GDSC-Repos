# Spring Data JPA를 활용한 REST API 설계

## 참고 사항
과제의 요구 사항은 딱 하나입니다. 상품 구매 서비스를 구현하세요!  
CRUD 기능만 REST API로 잘 구현하면 됩니다.  
설계는 처음부터 끝까지 모두 자유입니다.  

힌트를 주자면
* Customer (구매자)
* Item (상품)
* Order (주문)
* Controller, Service, Repository 계층

같은 상품 구매와 관련된 클래스들을 설계해보세요!
